# Linux 下 socket 编程 C语言 版

包含单线程版、多线程版、线程池版

```bash
.
├── README.md
├── socket
│   ├── build
│   ├── client.c
│   └── server.c
├── socket_thread
│   ├── build
│   ├── client.c
│   └── server_thread.c
└── socket_threadpool
    ├── build
    ├── client.c
    ├── server_threadpool.c
    ├── threadpool.c
    └── threadpool.h
```

- socket: 单线程版
- socket_thread: 多线程版
- socket_threadpool: 线程池版
